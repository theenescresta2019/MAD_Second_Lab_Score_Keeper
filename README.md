# MAD_Second_Lab_Score_Keeper
## The Badminton Score Keeper app
### Description
This app uses multiple Linear Layouts.
This app is designed to count the score of badminton game. You can set the winning point by yourself.
![Screenrecording-20201208-104346](https://user-images.githubusercontent.com/58808609/101443612-f623cc80-3945-11eb-9e04-c985966fc5da.gif)


If the score are draw in the game, the winning point will increase by 1. Any winning team should lead with min 2 points and reach winning point.
